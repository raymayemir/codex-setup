# Text And Typography

Use this reference when copy, headings, labels, metrics, tables, or dense text are driving the quality of the UI.

## Define A Limited Type Scale

Avoid accumulating random font sizes. Reuse a small handcrafted scale.

Example:
- `12, 14, 16, 18, 20, 24, 30, 36, 48`

The exact values can vary by product, but the set should feel intentional and limited.

## Pick Fonts For Fit And Legibility

- Neutral sans-serif families are safe and often correct for product UIs.
- Distinctive display or serif pairings are useful when the surface benefits from a stronger voice.
- Prefer families with enough weights and strong legibility at UI sizes.
- Do not force a fancy font into a dense tool just to be "interesting".

## Control Reading Width

- Long-form text is easiest to read when lines stay in a moderate range.
- For prose or explanatory copy, constrain width rather than letting paragraphs stretch across the page.
- Metrics, labels, short headlines, and dashboards can use different rules than body copy.

## Line Height Depends On Context

- Larger text usually wants tighter line-height.
- Smaller or longer text usually wants more room.
- Long lines need more line-height than short lines.

Use line-height as a readability tool, not as a single global constant.

## Align For Readability

- Left-align most UI text in left-to-right languages.
- Avoid centering long passages.
- Right-align numbers in tables and comparisons when it improves scanning.
- When mixing type sizes on one line, align them by baseline rather than optical centering.

## Links, Labels, And Letter-Spacing

- Not every link needs loud accent color, especially in surfaces where almost everything is clickable.
- Make labels quieter than the values they support when the value is what matters.
- Tighten large headlines slightly if they look loose.
- Add slight positive letter-spacing to all-caps text if legibility needs help.

## Quick Checks

- Can you name the few font sizes in use?
- Do headings, labels, and body copy feel like a system?
- Is long text easy to read without becoming too wide?
- Are metrics and tabular values aligned for scanning?
